# SoloFit

SoloFit is a mobile-first workout tracker that helps you log workouts, track sets and reps, and monitor your fitness progress.

## Features

-   User authentication
-   Create and track workout routines
-   Log exercises, sets, and reps
-   Mobile-first responsive design

## Tech Stack

-   Next.js
-   Supabase
-   PostgreSQL
-   Tailwind CSS
-   shadcn/ui
